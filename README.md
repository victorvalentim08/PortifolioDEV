## 🛠️ Tecnologias & Ferramentas

### Front-end
*   **Framework:** Next.js (App Router)
*   **Styling:** Tailwind CSS
*   **Animações:** Framer Motion (Interações suaves e micro-interações)
*   **Linguagem:** TypeScript

### Back-end & Data
*   **Linguagens:** Node.js, PHP, Python, C++
*   **Banco de Dados:** PostgreSQL, MySQL (Prisma ORM)

## 📁 Destaques do Projeto

*   **Timeline Dinâmica:** Apresentação cronológica da carreira, do estágio na Vicunha Têxtil à graduação atual.
*   **Filtro de Projetos:** Sistema funcional para categorizar projetos Acadêmicos, Pessoais e Comerciais.
*   **Glassmorphism UI:** Navbar e componentes com efeito de vidro e desfoque, seguindo tendências modernas de design.
*   **Totalmente Responsivo:** Experiência otimizada para Desktop e Mobile.
